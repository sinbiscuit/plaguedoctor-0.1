This is the current progress on Plaguedoctor, a class for Mewgenics which includes so far
- 44/50 spells
- 17/25 passives

Things the mod still needs 
- Ability icons for Spells/Passives
- Proper setup for unlocking the class and it's associated unlocks
- Miniboss
- Items for the class

You will need both Mewtator (mod loader) and Mewjector (dll loader).

https://www.nexusmods.com/mewgenics/mods/1
https://www.nexusmods.com/mewgenics/mods/218

At the moment the class unlocks through simply clearing the tutorial. You can manually unlock the class through debug console, though be wary if you have all 14 classes a 15th will crash. Related is a collar crash fix mod that can be run using Mewjector (enable dll loading in Mewtator and stuff Mewjector's stuff into your game folder).

https://github.com/hmrrrrr/CollarCrashFix

I HIGHLY recommend either playing on a fresh save file OR make a copy of your regular game file for this.
